# OC-S-ReportGenerator
Helper for OpenCore-Simplify to generate report for your hardware on any Linux distro

# Usage
1. Copy this repo.
```bash
git clone https://github.com/OneDevelopmentPL/OC-S-ReportGenerator.git
```
2. Go to project directory:
```bash
cd OC-S-ReportGenerator
```
3. Run the Python script: 
```bash
sudo python3 generate_report.py
```
MAKE SURE TO RUN IT WITH SUDO!

Then you get a file, called Report.json
Now you can use this json file for OpCore-Simplify
